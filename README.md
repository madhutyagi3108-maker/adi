# Aditya Chauhan — Portfolio

A stunning single-page portfolio built with pure **HTML + CSS + JS**.
No framework. No build step. No dependencies. Just upload and it works.

---

## 🚀 Deploy to Vercel (60 seconds)

### Step 1 — Push to GitHub
1. Create a new GitHub repo (e.g. `my-portfolio`)
2. Upload **all files** from this folder: `index.html`, `vercel.json`, `README.md`
3. Commit

### Step 2 — Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) → **Add New Project**
2. Import your GitHub repo
3. **Framework Preset**: select **Other** (not Next.js)
4. Click **Deploy** ✅

Your site will be live at `your-name.vercel.app` in under a minute.

## ✏️ How to Customize

Open `index.html` in any text editor and update:

| What to change | Where to find it |
|---|---|
| Name, tagline | `#hero` section |
| Bio text | `#about` section |
| Skills & percentages | `#skills` section, `data-w` attribute |
| Internship details | `#experience` section |
| Project cards | `#projects` section |
| Certifications | `#certifications` section |
| Contact info | `#contact` section + nav `.nav-cta` |
| Colors / fonts | `:root` CSS variables at top of `<style>` |
